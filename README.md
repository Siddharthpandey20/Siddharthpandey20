> **[Air-Quality Forecasting](https://github.com/Siddharthpandey20/Air_Quality_Forecasting)** · *solo* — a portfolio project written like a paper.

```text
  CPCB sensor data ──► 97–221 engineered features ──► stacked ensemble
       (4 stations)        (inductive bias)            (XGBoost · LightGBM · RidgeCV)
                                                              │
                                       ┌──────────────────────┴──────────────────────┐
                                  PM2.5 R² = 0.883                vs published benchmarks
                                  (4 pollutants + AQI)               +3.9% to +10.4%
```

- Includes an **explicit benchmark table** against published research — not a number in isolation.
- LSTM baseline included *specifically* to show why gradient-boosted trees win on this tabular data.
- Citation + reproduction sections — research-track rigor, not a tutorial follow-along.
